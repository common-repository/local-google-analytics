Enables you to host Google Analytics tracking script (analytics.js) locally on your webspace in order to increase website performance.

== Description ==
This plugin downloads analytics.js to your webspace in order to increase website performance (Pagespeed Insights). It adds a custom tracking snippet to your website with your porperty ID and your local analytics.js.


== Installation ==
1) Upload `Local Google Analytics` to the `/wp-content/plugins/` directory.
2) Login your WordPress backend.
3) Activate the plugin. That\'s it!
4) Go to `Settings` --> `Local Google Analytics`
